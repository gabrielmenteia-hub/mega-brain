```python
# pain_signals schema:
# url_hash: str (UNIQUE), url: str, title: str, source: str,
# niche_slug: str, score: int, extra_json: str, collected_at: str

def _url_hash(url: str) -> str:
    """SHA-256 do URL como chave de deduplicação."""
    import hashlib
    return hashlib.sha256(url.encode()).hexdigest()

# upsert pattern (sqlite-utils):
db["pain_signals"].upsert(
    {**signal, "url_hash": _url_hash(signal["url"])},
    pk="url_hash",
    alter=True,
)
```

- `collect_trends_signal(keyword: str, anchor: str, niche_slug: str) -> dict | None`
  - Instancia TrendReq(hl='pt-BR', tz=360) de `pytrends.request` (pacote: pytrends-modern)
  - build_payload([keyword, anchor], timeframe='now 7-d')
  - interest_over_time() com is_partial filtrado (última linha pode ser parcial)
  - peak_index = int(df[keyword].max()) — apenas o pico, não série completa
  - Retorna dict: {url_hash, url, title, source, niche_slug, score (=peak_index), extra_json (JSON com anchor, keyword), collected_at}
  - `url` para Trends: f"https://trends.google.com/trends/explore?q={keyword}" — único por keyword
  - Em Exception: log.error(..., alert='trends_ratelimited'), return None
  - Delay de 5-10s entre chamadas (usar `asyncio.sleep(random.uniform(5, 10))` após cada keyword)

- `collect_niche_trends(niche: dict, db_path: str) -> list[dict]`
  - Itera niche["keywords"] chamando collect_trends_signal com niche["radar"]["anchor_term"]
  - Persiste cada sinal em pain_signals via upsert (_url_hash pattern da migration)
  - Retorna lista de sinais coletados (exclui None)

Criar `mis/radar/reddit_collector.py`:
- `_sync_collect_reddit(niche: dict) -> list[dict]`
