- 'pending', 'running' — scan em andamento
- 'done' — scan concluiu (spy ainda não rodou ou foi pulado)
- 'spying' — spy em andamento (retorno imediato de run_manual_search não chega aqui normalmente)
- 'spy_done' — tudo concluído
- 'timeout', 'cancelled' — terminados com erro

    ```python
    # Re-spy: disparar spy para top-5 sem dossier ao abrir resultados antigos
    if session["status"] in ("done", "spy_done"):
        from collections import defaultdict
        db = request.app.state.db_path  # já está em db_path
        # Buscar produtos top-5 por plataforma sem dossier 'done'
        from mis.db import get_db as _get_db
        _db = _get_db(db_path)
        respy_rows = _db.execute(
            """
            SELECT ssp.product_id, ssp.platform_slug, ssp.rank_at_scan
              FROM search_session_products ssp
              LEFT JOIN dossiers d ON d.product_id = ssp.product_id
             WHERE ssp.session_id = ?
               AND (d.id IS NULL OR d.status != 'done')
             ORDER BY ssp.platform_slug, ssp.rank_at_scan ASC
            """,
            [session_id],
        ).fetchall()
        _top: dict[str, list] = defaultdict(list)
        for pid, pslug, rank in respy_rows:
            if len(_top[pslug]) < SPY_V3_TOP_N:
                _top[pslug].append({"id": pid, "rank": rank if rank is not None else 999})
        _products_to_respy = [p for ps in _top.values() for p in ps]
        if _products_to_respy:
            _task = asyncio.create_task(
                run_spy_batch(_products_to_respy, max_concurrent=SPY_V3_TOP_N)
            )
            register_task(session_id, _task)
            update_session_status(
                db_path, session_id, "spying",
                session.get("platform_statuses", {}),
                session.get("product_count", 0),
            )
            # Re-fetch session to reflect 'spying' status in the rendered page
            session = get_session(db_path, session_id)
    ```

    Nota: A variável `db_path` já está definida no início de search_results() como `request.app.state.db_path`. Não usar `db` como nome de variável (conflito com uso local existente).
