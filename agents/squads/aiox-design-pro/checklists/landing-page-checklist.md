# Landing Page Checklist

**ID:** DP-CL-001
**Versao:** 1.0
**Aplicacao:** Toda landing page criada ou revisada pelo Design Pro Squad
**Score minimo:** 75/100

---

## Como Usar

Para cada item: PASS (2pt) | PARCIAL (1pt) | FAIL (0pt)

Score final = (pontos obtidos / pontos possiveis) x 100

## Itens Veto (Blocking — Score zerado se falhar)

Independente do score geral, esses itens nao podem falhar:

| # | Item Veto |
|---|-----------|
| V1 | CTA presente na pagina |
| V2 | Garantia presente |
| V3 | Pagina carrega sem erro no mobile |
| V4 | Nenhuma claim sem evidencia ("melhor do mundo", "100% garantido" sem contexto) |
| V5 | Nao ha urgencia falsa (contador que reinicia, vagas sem numero) |
