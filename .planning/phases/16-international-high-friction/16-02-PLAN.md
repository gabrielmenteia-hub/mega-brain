> **Nota técnica — GumroadScanner e fetch_spa():**
> O CONTEXT.md (decisão locked) especifica "Usar `fetch_spa()` do BaseScraper" para o GumroadScanner.
> Na prática, GumroadScanner implementa `_scan_with_scroll()` que usa Playwright diretamente
> (não via `self.fetch_spa()`). Isso não contradiz a intenção da decisão — o objetivo era usar
> Playwright + playwright-stealth para renderizar a SPA. A razão técnica para não delegar a
> `fetch_spa()` é que o scroll loop **precisa do objeto `page`** para chamar
> `page.evaluate("window.scrollTo(...)")`, `page.wait_for_timeout()` e
> `page.query_selector_all()` a cada iteração. `fetch_spa()` encapsula o ciclo de vida completo
> do browser e retorna apenas o HTML final — sem expor o `page` para manipulação intermediária.
> `_scan_with_scroll()` adquire `PLAYWRIGHT_SEMAPHORE` diretamente (mesmo semaphore global),
> portanto a proteção contra OOM permanece intacta. AppSumoScanner, que não precisa de scroll
> interativo, usa `self.fetch_spa()` normalmente conforme a decisão locked.

Padrão de scroll loop (de RESEARCH.md — GumroadScanner deve implementar internamente):
```python
# GumroadScanner NÃO usa self.fetch_spa() diretamente — precisa do objeto page para scroll.
# fetch_spa() encapsula o ciclo de vida completo do browser e retorna apenas HTML final,
# sem expor o objeto page para chamar page.evaluate(), page.wait_for_timeout() e
# page.query_selector_all() a cada iteração do loop. Por isso _scan_with_scroll() gerencia
# o browser diretamente, mas adquire PLAYWRIGHT_SEMAPHORE explicitamente (mesmo semaphore
# global) para manter a proteção contra OOM. Decisão locked: "usar fetch_spa() do BaseScraper"
# — interpretada como "usar Playwright + stealth", não "delegar à função fetch_spa()".
# AppSumoScanner usa self.fetch_spa() normalmente (sem scroll loop).
async def _scan_with_scroll(self, url: str, limit: int = 50) -> str:
    from mis.base_scraper import PLAYWRIGHT_SEMAPHORE
    from playwright.async_api import async_playwright
    from playwright_stealth import Stealth as _PlaywrightStealth

    async with PLAYWRIGHT_SEMAPHORE:
        async with async_playwright() as pw:
            browser = await pw.chromium.launch()
            try:
                page = await browser.new_page()
                await _PlaywrightStealth().apply_stealth_async(page)
                await page.goto(url, wait_until="networkidle")
                prev_count = 0
                stable_rounds = 0
                for _ in range(20):  # max 20 scrolls
                    await page.evaluate("window.scrollTo(0, document.body.scrollHeight)")
                    await page.wait_for_timeout(1500)
                    items = await page.query_selector_all("[data-component-name='DiscoverProduct'], .product-card, article")
                    curr_count = len(items)
                    if curr_count >= limit:
                        break
                    if curr_count == prev_count:
                        stable_rounds += 1
                        if stable_rounds >= 2:  # 2 rounds sem crescimento = fim
                            break
                    else:
                        stable_rounds = 0
                    prev_count = curr_count
                content = await page.content()
            finally:
                await browser.close()
    return content
```
