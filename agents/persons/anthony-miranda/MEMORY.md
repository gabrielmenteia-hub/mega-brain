#### Frameworks Incorporados

**Dívida Inteligente (OPM Framework)**
- O que é: capital de terceiros alocado em ativos que geram renda maior que o custo da dívida
- Quando usar: sempre que quiser adquirir imóvel sem desembolso próprio
- Como aplicar: financiamento bancário, consórcio, seller financing, hipoteca sobre valorização
- Fonte: → chunk_AMD0F_008, chunk_AMD0F_009

**Princípio da Não-Venda**
- O que é: imóvel quitado nunca é vendido, exceto para comprar dois
- Quando usar: qualquer decisão de saída de ativo imobiliário
- Como aplicar: filtro automático — "vou comprar dois no lugar?" Se não, não vende
- Fonte: → chunk_AMD0F_005, AMD0F_INS_001

**Seleção Classe C (42-80m²)**
- O que é: foco em metragem menor para classe econômica com maior retorno proporcional
- Quando usar: decisão de qual imóvel comprar
- Como aplicar: relação aluguel/preço > que imóveis maiores; 60-70% da população brasileira
- Fonte: → chunk_AMD0F_022, AMD0F_INS_005

**Leilão como Laboratório**
- O que é: primeiro imóvel via leilão com 5% de entrada para aprender o jogo
- Quando usar: para quem está começando e quer minimizar risco
- Como aplicar: imóvel abaixo de mercado + financiamento + aluguel cobre parcela
- Fonte: → chunk_AMD0F_012, AMD0F_INS_004

### 📚 Anthony Miranda (AMCD6) — 2026-05-03

#### Frameworks Incorporados

**Buy-and-Hold Imobiliário (vs Buy-and-Flip)**
- O que é: comprar imóvel em leilão, alugar, repetir — jamais vender para realizar lucro
- Quando usar: qualquer decisão de saída de ativo adquirido em leilão
- Como aplicar: compra → arremata → aluga → repete. "Acabou, acabou, acabou."
- Fonte: → AMCD6_010, AMCD6_021, AMCD6_022

**Gestão Totalmente Terceirizada**
- O que é: equipe jurídica + assessor de leilão (Edwards) + administradora imobiliária (7-8%)
- Quando usar: a partir do segundo imóvel
- Como aplicar: jurídico pré-analisa, Edwards filtra leilões, administradora gera aluguel e resolve problemas
- Fonte: → AMCD6_007, AMCD6_020

**Aquisição via Dívida do Inadimplente (MRV/Populares)**
- O que é: pagar a dívida bancária do proprietário que não consegue mais pagar, ficando com o imóvel
- Quando usar: proprietário inadimplente que só quer o nome limpo
- Como aplicar: assume a dívida → imóvel passa para você → aluga. Custo de entrada ≈ zero.
- Fonte: → AMCD6_023

**Dosagem de Aquisição pelo Estoque**
- O que é: só comprar mais quando estiver com 1-2 unidades restantes para alugar
- Quando usar: sempre que avaliar compra de novo imóvel
- Como aplicar: checar estoque → se > 2 vagas abertas, aguardar; se ≤ 2, comprar
- Fonte: → AMCD6_007
