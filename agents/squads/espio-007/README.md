# Espião 007 Squad

> Espionagem sistemática de infoprodutos e produtos de alto ticket no mercado americano.

## Missão

Rastrear nichos em alta, mapear funis completos (VSL → OTO → Downsell), extrair e analisar copy, desestruturar ofertas e entregar briefs acionáveis para adaptação ao mercado brasileiro.

## Plataformas Monitoradas

| Plataforma | Foco |
|---|---|
| ClickBank | Gravity Score alto |
| Digistore24 (US) | Produtos emergentes |
| JVZoo | EPC + volume de afiliados |
| Warrior Plus | Lançamentos e PLFs |
| ThriveCart | Produtos de alto ticket |

## Agentes

| Agente | Comando | Função |
|---|---|---|
| Bond | `/007-bond` | Orquestrador — coordena a missão completa |
| Scout | `/007-scout` | Rastreia nichos e produtos em alta |
| Funnel Mapper | `/007-funnel-mapper` | Mapeia funis VSL → OTO → Downsell |
| Copy Analyst | `/007-copy-analyst` | Extrai e analisa copy de alta conversão |
| Offer Decoder | `/007-offer-decoder` | Desestrutura stacks de oferta |
| Brief Writer | `/007-brief-writer` | Gera briefs para adaptação no BR |

## Fluxo de Missão

```
1. /007-bond       → Define alvo e escopo da missão
2. /007-scout      → Identifica e rankeia produtos (ClickBank, JVZoo, etc.)
3. /007-funnel-mapper → Documenta o funil completo
4. /007-copy-analyst  → Analisa VSL e sales page
5. /007-offer-decoder → Desestrutura a oferta
6. /007-brief-writer  → Gera brief de adaptação BR
7. /007-bond       → Consolida e entrega relatório final
```

## Uso Rápido

```
# Missão completa (recomendado)
/007-bond

# Análise pontual
/007-scout          → Encontrar produto para espionar
/007-funnel-mapper  → Mapear funil de produto específico
/007-copy-analyst   → Analisar copy de uma sales page
/007-offer-decoder  → Desestruturar uma oferta
/007-brief-writer   → Gerar brief de adaptação BR
```
