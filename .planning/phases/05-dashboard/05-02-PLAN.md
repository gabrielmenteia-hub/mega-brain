Mas get_db() aceita db_path:
```python
def get_db(db_path: str) -> sqlite_utils.Database:
    db = sqlite_utils.Database(db_path)
    db.execute("PRAGMA journal_mode=WAL")
    db.execute("PRAGMA foreign_keys=ON")
    return db
```

       - create_alert(db_path: str, product_id: int, platform_slug: str, niche_slug: str, position: int) -> int

         UPDATE alerts SET seen=1 WHERE id=alert_id. Retorna True se rowcount>0.

         DELETE FROM alerts WHERE expires_at < now. Retorna contagem deletada.
