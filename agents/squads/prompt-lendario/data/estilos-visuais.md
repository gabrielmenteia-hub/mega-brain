# Biblioteca de Estilos Visuais

> Referência de tokens por estética para prompts de imagem e vídeo.

---

## Estilos Fotográficos

### Editorial / Fashion
```
editorial photography, fashion photography, vogue aesthetic,
high fashion, model posing, dramatic shadows, high contrast,
luxury brand, clean composition
```

### Produto Premium
```
product photography, studio lighting, white/black seamless background,
soft box lighting, reflective surface, luxury packaging,
hyperrealistic, commercial photography
```

### Documentary / Raw
```
documentary photography, candid, photojournalism style,
natural light, grain, authentic moment, street photography,
35mm film, Leica aesthetic
```

### Cinematográfico
```
cinematic photography, anamorphic lens, lens flare,
film grain, color grading, dramatic lighting,
movie still, wide angle, shallow depth of field
```

## Paletas de Cor

### Luxury / Premium
```
dark moody, rich blacks, gold accents, deep burgundy,
muted tones, desaturated, monochromatic, elegant
```

### Energia / Impacto
```
vibrant colors, high saturation, neon accents,
bold contrast, electric blue, orange pop
```

### Clean / Minimalista
```
white space, neutral tones, pastel, soft shadows,
airy, light and bright, Scandinavian aesthetic
```

### Quente / Orgânico
```
warm tones, golden hour, amber, terracotta,
earth tones, warm shadows, organic textures
```

| Tipo | Tokens |
|------|--------|
| Dramática | `dramatic lighting, chiaroscuro, single key light, hard shadows` |
| Suave | `soft box lighting, diffused light, no harsh shadows, beauty lighting` |
| Natural | `golden hour, natural light, window light, outdoor lighting` |
| Rim Light | `rim lighting, backlit, hair light, silhouette effect` |

| Cinematográfica | `cinematic lighting, motivated light, color gels` |

## Ângulos e Composição

| Ângulo | Token |
|--------|-------|
| Olho do nível | `eye level shot` |
| Baixo (heroico) | `low angle shot, worm's eye view` |
| Alto (overview) | `high angle shot, bird's eye view` |
| Aéreo | `aerial view, drone shot, top down` |
| Close extremo | `extreme close-up, macro` |
| Retrato | `medium close-up, portrait framing` |
| Ambiental | `wide establishing shot, environmental portrait` |

## Referências Cinematográficas

### Realistas / Elegantes
- Roger Deakins (Blade Runner 2049, 1917)
- Emmanuel Lubezki (The Revenant, Gravity)

### Coloridos / Estilizados
- Wes Anderson — pastel, simetria, flat lay aesthetic
- Wong Kar-Wai — color gels, moody, blurred motion

### Comercial / Produto
- Peter Lindbergh (fashion documentary)
- Annie Leibovitz (editorial portraiture)

**Como usar em prompt:**
```
in the style of Roger Deakins, cinematography inspired by Blade Runner 2049
Wes Anderson aesthetic, symmetrical composition, pastel color palette
```
